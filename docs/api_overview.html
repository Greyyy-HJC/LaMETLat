lametlat API Overview

A compact guide to the package modules, the public functions they expose, and the common analysis tasks they support in LaMET lattice-QCD workflows.

Minimal data flow

Read or prepare data, resample it, fit a model, then plot the result.

from lametlat.correlators import bootstrap, bs_ls_avg
from lametlat.ground_state import pt2_fit
from lametlat.plotting import pt2_plot

samples = bootstrap(raw_pt2, n_samples=200, axis=0)
pt2_gv = bs_ls_avg(samples, axis=0)
fit = pt2_fit(pt2_gv, tmin=3, tmax=12, Lt=32)
pt2_plot([pt2_gv], fit_results=fit, fit_tmin=3, fit_tmax=12)
